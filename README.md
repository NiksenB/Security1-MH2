# disys_miniproject3

You need to run these commands, each in a new terminal.

```
go run .\server\server.go
go run .\client\client.go
go run .\client\client.go
```

Name the two clients whatever, then do the following exchange:

### client 1

```
send commitment
```

### client 2

```
send roll
```

### client 1

```
reveal commitment
```

### client 2

```
validate commitment
```
